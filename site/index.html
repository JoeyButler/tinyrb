tinyrb

A tiny Ruby VM for you to love and play with

tinyrb is the smallest Ruby VM to date (64K). It wont run all your Ruby code, but it's pretty fast.

Goals

  • Portable
  • Embedable
  • Fast
  • Low memory usage
  • Small code base, easy to read
  • Fun and awesome

Try it!

git clone git://github.com/macournoyer/tinyrb.git
cd tinyrb; make
./tinyrb your_script_that_will_crash.rb

Contribute

To discuss and share about tinyrb, comment on the commits feed on GitHub.

VM specs

  • Register based
  • Monomorphic inline cache
  • Direct threaded dispatch
  • Using the Boehm GC (for now)
  • Very small < 64K
